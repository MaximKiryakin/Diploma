# Академические работы

Репозиторий содержит материалы выпускных квалификационных работ, выполненных во время обучения в МГУ имени М. В. Ломоносова. Основные материалы разделены по уровню обучения: бакалавриат и магистратура.

## Структура репозитория

```text
.
├── bachelor/   # Материалы бакалаврской выпускной работы
└── masters/    # Материалы магистерской диссертации
```

## `bachelor/`

Директория содержит материалы бакалаврской ВКР, посвященной моделированию распространения респираторной инфекции с использованием компартментно-сетевых моделей.

Основное содержимое:

- `main.py` и `main.ipynb` - код и ноутбук с реализацией модели, загрузкой данных и вычислительными экспериментами.
- `input/` - исходные данные для построения популяции и сетей контактов: домохозяйства, школы, предприятия, возрастно-половое распределение.
- `utils/` - вспомогательные функции для чтения и подготовки данных.
- `text/` - LaTeX-исходники текста работы и изображения для нее.
- `presentation/` - материалы презентации, включая исходники, PDF и иллюстрации.

Внутри работы рассматриваются сети контактов для городских и сельских популяций, SIR-подобная динамика и сравнение моделирования на реалистичных и случайных сетях.

## `masters/`

Директория содержит материалы магистерской диссертации на тему моделирования управления кредитным портфелем с учетом макроэкономических условий.

Основное содержимое:

- `README.md` - подробное описание магистерского проекта, его методологии, структуры и быстрого старта.
- `main.ipynb` и `main.html` - основной расчетный ноутбук и его HTML-версия.
- `requirements.txt` - зависимости Python-проекта.
- `utils/` - прикладная логика проекта: загрузка данных, модель кредитного риска, оптимизация портфеля, визуализация и конфигурация.
- `data/` - локальные данные и резервные копии для расчетов.
- `logs/` - логи выполнения и сохраненные графики.
- `text/` - LaTeX-исходники и PDF диссертации.
- `paper/` - материалы статьи по теме работы.
- `presentation/` - презентационные материалы.

Магистерская часть объединяет оценку вероятности дефолта по модели Мертона, прогнозирование макроэкономических факторов, портфельную оптимизацию и бэктестинг стратегий управления кредитным портфелем.
