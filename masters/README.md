## Аннотация

Работа посвящена разработке и программной реализации **гибридного алгоритма динамического
управления кредитным портфелем**, который объединяет структурную модель оценки вероятности
дефолта, эконометрическое прогнозирование макропоказателей и многокритериальную портфельную
оптимизацию в единый итеративный конвейер. Подход позволяет перейти от реактивного к
превентивному управлению кредитным риском и минимизировать кредитный и рыночный риск при
заданном уровне риск-скорректированной доходности.

На кредитный риск приходится около 70 % совокупного риск-профиля российского банковского
сектора, при этом классические статические подходы к оценке кредитоспособности не учитывают
динамику макроэкономических шоков. Предлагаемый алгоритм закрывает этот пробел, адаптируя
модель Мертона к российскому рынку и интегрируя её с прогнозом ключевых макрофакторов
(инфляция, ключевая ставка, безработица, курс USD/RUB).

## Ключевые компоненты

| Блок | Содержание |
|------|------------|
| **Оценка дефолта** | Структурная модель Мертона: восстановление стоимости и волатильности активов, расчёт Distance-to-Default и PD по рыночным данным |
| **Эконометрика** | Pooled OLS на панельных данных (1092 наблюдения): оценка чувствительности DD к макрофакторам с тестами значимости |
| **Прогноз макросреды** | Сценарное прогнозирование моделями **VAR**, **SARIMAX** и **Prophet** с walk-forward сравнением по MAE / RMSE / MAPE |
| **Оптимизация** | Унифицированная многокритериальная модель с тремя мерами риска: волатильность (**Mean-EL**), **CVaR** и **Risk Parity**; расширение до распределения абсолютных сумм заявок (amount-based) с целевой функцией RAROC и секторальными лимитами |
| **Бэктестинг** | Walk-forward симуляция четырёх стратегий на 12-месячном горизонте с учётом транзакционных издержек |

---

## Научная новизна

- **Интегрированный конвейер** «Мертон → макропрогноз → оптимизация» в едином итеративном цикле.
- **Унифицированная оптимизационная модель**, поддерживающая три меры риска в одной постановке,
  что обеспечивает методологическую сопоставимость результатов.
- **Обобщение Risk Parity** на случай кредитного риска через совокупный рисковый вклад $CRC_i$,
  объединяющий рыночную и кредитную компоненты.
- **Amount-based постановка**: распределение абсолютных сумм кредитных заявок с максимизацией
  RAROC, учётом кумулятивной PD на срок кредита, индивидуальных $LGD_i$ и секторальных лимитов.

## Результаты бэктеста

Сравнение стратегий на 12-месячном walk-forward горизонте (российский рынок в фазе снижения):

| Стратегия | Нетто-доход, % | EL, % | Волатильность, % | Ребаланс. | Sharpe |
|-----------|:---:|:---:|:---:|:---:|:---:|
| Passive (Equal, B&H) | −14.01 | 0.29 | 14.26 | 0 | 0.21 |
| Mean-EL | −12.19 | 0.24 | 13.82 | 5 | 0.26 |
| CVaR | −11.48 | 0.26 | 13.45 | 6 | 0.29 |
| **Risk Parity** | **−10.94** | **0.22** | **12.98** | **4** | **0.33** |

Все активные стратегии превзошли пассивный бенчмарк по нетто-доходу и коэффициенту Шарпа.
Лучший результат — **Risk Parity**: минимальные волатильность и ожидаемые потери при
максимальном Sharpe и умеренном числе ребалансировок.

## Быстрый старт

Откройте `main.ipynb` и запустите ячейки последовательно, либо используйте API напрямую.
Класс `Portfolio` реализует fluent-интерфейс: методы возвращают `self` и объединяются в цепочку.

```python
from utils.setup import setup_environment
from utils.portfolio import Portfolio

setup_environment()

tickers_list = [
    "GAZP", "LKOH", "ROSN",   # Нефтегазовый сектор
    "SBER", "VTBR", "MOEX",   # Финансовый сектор
    "GMKN", "NLMK", "RUAL",   # Металлургия
    "MTSS", "RTKM", "TTLK",   # Телекоммуникации
    "MGNT", "FESH",           # Розничная торговля
]

calc = Portfolio(
    dt_calc="2025-09-30",
    dt_start="2019-03-31",
    stocks_step=8,
    tickers_list=tickers_list,
)

# 1. Загрузка данных: рыночные котировки, мультипликаторы, макрофакторы
calc = (
    calc
    .load_stock_data(use_backup_data=True)
    .load_multipliers(use_backup=True)
    .load_macro_data()
    .create_portfolio()
    .add_dynamic_features()
    .add_merton_pd()          # 2. Оценка PD по модели Мертона
)

# 3. Прогноз макрофакторов и анализ значимости
calc = (
    calc
    .forecast_macro(horizon=3, models=["var", "sarimax", "prophet"], target_col="DD")
    .analyze_macro_dd_significance(verbose=True)
)

# 4. Оптимизация портфеля (amount-based) и сравнение стратегий
calc = calc.optimize_portfolio_with_amounts(
    budget=20e9,
    max_sector_share=0.3,
)
calc = calc.compare_amount_strategies(
    strategies=["mean_el", "cvar", "risk_parity"],
    rebalance_threshold=0.02,
    transaction_cost=0.001,
)
```

## Источники данных

| Показатель | Источник |
|------------|----------|
| Котировки акций (дневные) | Московская биржа (MOEX) |
| Финансовая отчётность (МСФО) | Публичные отчёты эмитентов |

| Уровень безработицы | Росстат |

Эмпирическая база охватывает 14 публичных эмитентов пяти секторов экономики
(нефтегазовый, финансовый, металлургический, телекоммуникационный, потребительский)
за период с 2019 года.
